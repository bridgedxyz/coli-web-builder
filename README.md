# web-builder

jsx syntex tree builder for web applications. react, vue, svelte, etc..

> A [CoLI](coli.codes) Based Technology

## installation

```shell
yarn add @coli.codes/web-builder
```

## platforms

- react
- vue
- svelte
- css
- sass
- less
- mui
- tailwind css
- bootstrap
- [reflect](https://reflect-ui.com)

## Usage

```
new Component()
```

```tsx
<Component style={{ color: CSS.black }}>
  <Child />
</Component>

// <>
```
